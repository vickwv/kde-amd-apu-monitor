# Translation Information

This directory contains translation files for the AMD APU Monitor widget.

## Supported Languages:
- English (en)
- Simplified Chinese (zh_CN)
- Traditional Chinese (zh_TW)

## Translation Process:
1. Extract strings: Run extract-messages.sh to extract translatable strings from QML files
2. Translate: Edit the .po files in each language directory
3. Compile: The .po files will be compiled to .mo files during installation

## Directory Structure:
- en/ - English translations
- zh_CN/ - Simplified Chinese translations  
- zh_TW/ - Traditional Chinese translations
